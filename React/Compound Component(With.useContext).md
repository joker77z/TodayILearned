# 요구사항

- Shuffle 버튼을 누르면 랜덤 숫자가 나온다.
- 랜덤 숫자의 min, max를 정할 수 있다.
- reset버튼을 누르면 0으로 숫자가 초기화된다.
- input 요소에 랜덤 숫자가 나온다.
